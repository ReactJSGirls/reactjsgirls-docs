# Starter JSX

## Let's add the starter JSX

In our `index.js` find the `App` class and its `render` method. It currently returns this:

<Code code={`
return (
  <div className="App">
    <h1>Hello CodeSandbox</h1>
    <h2>Start editing to see some magic happen!</h2>
  </div>
)
`} />

Delete all those lines and replace them with the following JSX:

<Code code={`
return (
  <main>
    <section className="cat-wrapper">
      <h1>Choose your favorite cats</h1>
      <section className="cat-container">
        <figure>
          <img
            className="cat-image"
            src="https://catis.life/placeholder/400"
            alt="A awesome akita cat"
          />
        </figure>
      </section>
    </section>
  </main>
)
`} />

You should now have something like:

<img src={starter} style={{ maxWidth: '100%' }} />
