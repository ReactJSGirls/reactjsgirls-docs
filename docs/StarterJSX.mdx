# Starter JSX

## Let's add the starter JSX

In our `index.js` find the `App` class and its `render` method. It currently returns this:

<Code code={`
return (
  <div className="App">
    <h1>Hello CodeSandbox</h1>
    <h2>Start editing to see some magic happen!</h2>
  </div>
)
`} />

Delete all those lines and replace them with the following JSX:

<Code code={`
return (
  <main>
    <section className="dog-wrapper">
      <h1>Choose your favorite dogs</h1>
      <section className="dog-container">
        <figure>
          <img
            className="dog-image"
            src="https://images.dog.ceo/breeds/akita/Akita_Dog.jpg"
            alt="A great akita dog"
          />
        </figure>
      </section>
    </section>
  </main>
)
`} />
